# Quick-Calc (swe-testing-assignment)

Quick-Calc is a lightweight calculator that supports addition, subtraction, multiplication, division (with graceful division-by-zero handling), and a Clear (C) reset. The main goal is demonstrating clean code, version control, and a multi-layer testing strategy.

## Setup Instructions
### Requirements
- Python 3.10+ recommended

### Installation
```bash
python -m venv .venv
# Windows: .venv\Scripts\activate
# macOS/Linux: source .venv/bin/activate
pip install -r requirements.txt
